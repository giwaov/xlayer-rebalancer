How It Works

Fully autonomous rebalancing

The agent monitors your X Layer portfolio and executes DEX swaps when allocation drifts beyond your threshold.

🎯
Define Targets
Set allocation weights in config.py. Example: 60% OKB, 40% USDT.
🔍
Monitor Drift
Agent polls balances and prices every 5 minutes, computing real-time drift.
⚡
Auto Rebalance
When drift exceeds threshold, precise DEX swaps execute via OnchainOS.
🛡️
Security First
Every token is security-scanned before trading. Honeypots are blocked.
🤖
OnchainOS Powered
Built on OKX's OnchainOS - wallet, swap, market, and security skills natively.
📦
Zero Dependencies
Pure Python stdlib. No pip installs. Just the OnchainOS CLI.
🔒
Paper Mode
DRY_RUN=True by default. Simulate everything before risking real funds.
Quick Start

Run the rebalancer

The agent runs locally as a Python script. OnchainOS CLI manages your wallet securely.

01
Install OnchainOS CLI
Install the CLI and authenticate your wallet. Keys never appear in code.
# Install npm install -g onchainos # Authenticate onchainos wallet login # Verify onchainos wallet address
02
Clone & Configure
Clone the repo and edit config.py to set your allocation targets.
git clone https://github.com/giwaov/xlayer-rebalancer cd xlayer-rebalancer
ParameterDefaultWhat it does
DRY_RUNTruePaper mode - no real swaps
DRIFT_THRESHOLD5.0%Min drift to trigger rebalance
CHECK_INTERVAL300sPoll interval
TARGET_PORTFOLIO60/40OKB/USDT split
03
Run in Paper Mode
Start the agent. It fetches real prices and balances but only simulates swaps.
# Start (paper mode is default) python scripts/agent.py # Dashboard at http://localhost:3197 # Checks balances every 5 min # Computes drift vs targets # Logs everything to events.json
04
Go Live
Set DRY_RUN = False in config.py and restart. The agent executes real swaps.
# In scripts/config.py: DRY_RUN = False # Restart python scripts/agent.py # Security scans every token # Max $50 per swap # 0.5% slippage protection
