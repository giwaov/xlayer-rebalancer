How It Works

Set it and forget it

The agent runs autonomously, monitoring your portfolio around the clock and executing precise swaps to maintain your target allocation.

🎯
Define Targets
Set your ideal allocation across tokens - for example, 60% OKB and 40% USDT. Fully configurable.
🔍
Monitor Drift
The agent polls balances and market prices every 5 minutes, calculating allocation drift in real time.
⚡
Auto Rebalance
When drift exceeds your threshold, the agent executes precise DEX swaps via OnchainOS to restore balance.
🛡️
Stay Secure
Every token is security-scanned before trading. Honeypots and malicious contracts are automatically blocked.
🤖
OnchainOS Powered
Built on OKX's OnchainOS infrastructure, using wallet, swap, market, and security skills natively.
💸
Zero Dependencies
Pure Python with zero pip installs. The agent uses only the standard library and OnchainOS CLI.
🔒
Paper Mode First
Starts in paper trading mode by default. Simulate and verify before going live with real funds.
